# Engineering-Analytics-Data-Platform-for-CAD-PLM-Systems

### Project Description :

#### Engineering Analytics Data Platform (CAD/PLM)

#### An end-to-end data pipeline ingesting CAD and PLM engineering data using Azure Data Factory.

#### Processed large design datasets using Databricks and Apache Spark.

#### Implementation of  Medallion architecture (Bronze/Silver/Gold) using Delta Lake.

#### Developed analytics datasets for engineering productivity, component reuse, and change management KPIs.
